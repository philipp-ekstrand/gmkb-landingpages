# GMKB Recruiting-Landingpages

Conversion-optimierte Recruiting-Landingpages fuer die GMKB (Gemeinnuetzige Medizinzentren Koeln/Bonn).

## Stack

- HTML5 + Vanilla CSS + Vanilla JS + PHP
- Kein Framework, keine Dependencies
- Integration: WordPress Custom Page + Code Snippets Plugin

## Landingpages

1. **Facharzt Paediatrie (m/w/d)** – `/pages/paediatrie/`
2. **Ergotherapeut*in / Logopaedin (m/w/d)** – `/pages/therapeuten/`
3. **Danke-Seite** – `/pages/danke/`
